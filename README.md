# Ruby on Rails 8 Starter Kit

This is a starter kit for Ruby on Rails 8 applications. It includes the following:

- Ruby 3.0.2
- Rails 8.0.0
- Tailwind CSS 2.2.19
- User authentication with built-in Rails authentication
- Role Based Authorization with Pundit
- Admin Dashboard
